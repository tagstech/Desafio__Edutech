Javascript: Programando a Orientação a Objetos


    Com esse curso de JavaScript você:
  • Domina o paradigma de programação mais usado no mercado
  • Entende o que são referências e objetos
  • Usa atributos, métodos da instância e da classe
  • Define objetos através de construtores
  • Aprende sobre encapsulamento


    • A primeira impressão


    Com base neste curso, serão apresentados abaixo alguns exemplos como forma de demonstração



    Classes


    Uma classe do JavaScript é um tipo de função. As classes são declaradas com a palavra-chave class ,esta função é como um molde de dados.


    Atributos


    Os Atributos em Programação Orientada a Objetos são os elementos que definem a estrutura de uma classe. Os atributos também são conhecidos como variáveis de classe, e podem ser divididos em dois tipos básicos: atributos de instância e de classe. Os valores dos atributos de instância determinam o estado de cada objeto.



    class Nome {
    Aqui ficam os atributos (também chamados de "campos" ou "propriedades");
    }
